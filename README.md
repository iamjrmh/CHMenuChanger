# CHMenuChanger  by JURMR

Change the menu backgrounds in Clone Hero without touching game files manually.

---

## Download

1. Go to the Releases page on this GitHub repository
2. Click the latest release
3. Under "Assets", download **CHMenuChanger.zip**
4. Extract the ZIP anywhere on your PC — your Desktop, a games folder, wherever you like

---

## Launch

Open the extracted folder and double-click **CHMenuChanger.exe**

That's it. No installer, no Python, nothing else to install.

---

## Important — Clone Hero Launcher

The Clone Hero launcher resets your game files back to default after every launch, which will undo any background changes made with this tool.

To prevent this, set up your install manually:

1. Install Clone Hero through the launcher as normal
2. Move that install folder somewhere else on your PC
3. In the launcher settings, remove the old install path
4. Point it at your new location instead

Once set up this way the launcher will no longer overwrite your files.

---

## How to use

1. Launch **CHMenuChanger.exe**
2. Click **Browse** and select your **Clone Hero_Data** folder
   - Default location: `Documents\Clone Hero\Clone Hero_Data`
3. Click **Load & Scan** — the app will find all background textures automatically
4. Click a background name in the list on the left to preview it
5. Create a new profile with **+ New** (the Default profile is read-only)
6. Click **Choose Replacement** and pick your image (PNG or JPG, 1920×1080 minimum)
7. Click **Apply & Save** to write the change directly into Clone Hero
8. Restart Clone Hero to see the new background in-game

To undo everything, click **Restore Backups** — the app automatically backs up your original files the first time you scan a folder.
